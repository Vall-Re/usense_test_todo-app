# TodoApp

## Список справ (To-Do List)

### Завдання:

Створити простий веб-додаток для ведення списку справ.

### Опис:

- Користувач може додавати нові завдання у список.
- Реалізувати можливість відзначати завдання як виконані та видаляти їх.
- Використати публічний API, наприклад, JSONPlaceholder, для створення та збереження завдань. Це дозволить симулювати взаємодію з сервером та перевірити роботу додатку при роботі з реальним бекендом.

### Додаткові умови:
- Додати можливість сортувати завдання за пріоритетом (наприклад, низький, середній, високий).
- Реалізувати фільтрацію завдань: показувати лише виконані, невиконані або всі завдання.
- Додати можливість редагувати завдання після їх створення.

## Angular Todo App

Функціональний менеджер завдань, побудований на Angular. Проєкт демонструє роботу з API, фільтрацію даних, керування станами компонентів та сучасну верстку.

- **[GitHub repository]**(https://github.com/Vall-Re/usense_test_todo-app)
- **[Demo Link]**(https://usense-test-todo-app.vercel.app/)

### Основні можливості

- **Завантаження даних:** Отримання початкового списку справ із зовнішнього API (JSONPlaceholder).

- **Керування завданнями:** Створення, редагування та видалення тасок.

- **Система пріоритетів:** Кожне завдання має свій рівень пріоритету (Low, Medium, High), що візуально відображається кольоровими рамками зліва.

- **Розумне редагування:** Можливість редагувати текст завдання подвійним кліком (лише для активних справ). Виконані завдання захищені від випадкових змін.

- **Фільтрація:** Швидке сортування за статусом (All, Active, Completed).

### Технологічний стек

- **Framework:** Angular
- **Language:** TypeScript
- **Styling:** CSS3
- **Deployment:** Vercel
